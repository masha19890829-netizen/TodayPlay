# TodayPlay

## Short Description
Genera una ruta para parejas, amigos o familias en 30 segundos.

## Full Description
TodayPlay convierte necesidades de relación, lugares de la ciudad, planificación de rutas, tareas de interacción, accesos a mapas, check-ins y puntos en una tarjeta de ruta fácil de ejecutar. Elige con quién vas, ciudad, tiempo, presupuesto, transporte y preferencias, y recibe una ruta con paradas, razones de recomendación, coste estimado, ideas para fotos, consejos de seguridad, tareas de interacción y una tarjeta final.

Esta versión usa solo datos locales curados y datos mock. No extrae contenido de Xiaohongshu, Douyin, Meituan, Amap ni otras plataformas de terceros, no muestra imágenes no autorizadas y no guarda API keys dentro de la app. Los accesos de mapa abren una app externa o el navegador mediante acciones del sistema.

## Release Notes
Se agregó posicionamiento global, textos multilingües de inicio y tienda, ciclo de ruta, puntos por check-in, historial local e integración cliente reservada para Google Play Billing.
