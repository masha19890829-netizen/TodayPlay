# 오늘 뭐 하고 놀까

## Short Description
커플, 친구, 가족을 위한 오늘의 루트 퀘스트를 30초 안에 생성합니다.

## Full Description
「오늘 뭐 하고 놀까」는 관계 요구, 도시 장소, 이동 루트, 인터랙션 미션, 지도 이동, 체크인, 리워드 포인트를 실행 가능한 루트 카드로 정리하는 앱입니다. 함께 가는 사람, 도시, 시간, 예산, 이동 방식, 분위기를 선택하면 장소, 추천 이유, 예상 비용, 사진 아이디어, 주의 사항, 인터랙션 미션, 완료 카드를 포함한 루트 퀘스트를 제공합니다.

현재 버전은 로컬 큐레이션 데이터와 mock 데이터만 사용합니다. Xiaohongshu, Douyin, Meituan, Amap 등 제3자 플랫폼을 스크래핑하지 않고, 승인되지 않은 이미지를 표시하지 않으며, 앱 안에 API Key 를 저장하지 않습니다. 지도 진입은 외부 지도 앱 또는 브라우저를 여는 방식입니다.

## Release Notes
글로벌 제품 방향, 다국어 홈/상점 문구, 루트 퀘스트 흐름, 체크인 포인트, 로컬 히스토리, Google Play Billing 클라이언트 연동 준비를 추가했습니다.
