# 今天怎麼玩

## Short Description
30 秒生成一條適合情侶、朋友、家人的今日路線副本。

## Full Description
「今天怎麼玩」把關係需求、城市地點、路線規劃、互動任務、地圖跳轉與打卡積分整理成一張可執行的今日路線卡。使用者選擇同行關係、城市、時間、預算、交通方式與偏好後，可以獲得包含地點、推薦理由、預估花費、拍照建議、避坑提示、互動任務與通關卡的路線副本。

目前版本使用本地精選與 mock 資料，不抓取小紅書、抖音、美團、高德等第三方平台內容，不展示未授權圖片，不在客戶端保存 API Key。地圖入口會透過系統跳轉開啟外部地圖 App 或瀏覽器。
