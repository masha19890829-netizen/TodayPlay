# 今日どう遊ぶ？

## Short Description
カップル、友達、家族向けの今日のルートクエストを 30 秒で作成。

## Full Description
「今日どう遊ぶ？」は、相手との関係、街のスポット、ルート計画、交流タスク、地図への移動、チェックイン、ポイントを、実行しやすいルートカードにまとめるアプリです。同行者、都市、時間、予算、交通手段、気分を選ぶだけで、立ち寄り地点、推薦理由、予算目安、写真アイデア、注意点、交流タスク、完了カードを含むルートクエストを生成します。

現在のバージョンはローカルの厳選データと mock データのみを使用します。小紅書、抖音、美団、高徳地図などの第三者プラットフォームをスクレイピングせず、未許可画像を表示せず、アプリ内に API Key を保存しません。地図入口は外部地図アプリまたはブラウザを開きます。
