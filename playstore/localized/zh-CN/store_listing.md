# 今天怎么玩

## Short Description
30 秒生成一条适合情侣、朋友、家人的今日路线副本。

## Full Description
「今天怎么玩」把关系需求、城市地点、路线规划、互动任务、地图跳转和打卡积分整理成一张可执行的今日路线卡。用户选择同行关系、城市、时间、预算、交通方式和偏好后，可以获得一条包含地点、推荐理由、预计花费、拍照建议、避坑提示、互动任务和通关卡的路线副本。

当前版本使用本地精选与 mock 数据，不抓取小红书、抖音、美团、高德等第三方平台内容，不展示未经授权图片，不在客户端保存 API Key。地图入口通过系统跳转打开外部地图应用或浏览器。
