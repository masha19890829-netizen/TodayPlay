# V0.9.32 分享闭环增强更新报告

日期：2026-06-11  
版本：`0.9.32 / versionCode 50`  
负责人视角：项目负责人调配产品、市场调研、开发  
本轮主题：把分享从“截图保存”推进到“邀请参与 + 通关传播”

## 背景

产品经理和市场调研都指出，TodayPlay 的分享不应该只是展示一张结果图，而应该区分两种社交动作：

- 出发前：把路线作为邀请发给对方，让对方参与选择。
- 完成后：把通关结果作为关系记忆和社交传播发出去。

## 已交付

1. 分享页按完成状态调整优先级
   - 未完成路线：优先展示邀请卡。
   - 已完成路线：优先展示通关卡。
   - 这样用户进入分享页时，默认看到更适合当下状态的分享动作。

2. 通关卡新增系统分享
   - 新增“分享通关结果”按钮。
   - 分享内容包括：
     - 路线标题
     - 通关称号
     - 任务完成进度
     - 地点打卡进度
     - 获得积分
     - 关键词
     - 完成总结

3. 多语言分享文案补齐
   - 邀请卡和通关卡分享文本覆盖：
     - 简体中文
     - 繁体中文
     - 英文
     - 日文
     - 韩文
     - 西班牙语

4. 版本与项目材料同步
   - App 版本升到 `0.9.32 / versionCode 50`。
   - 同步发布计划、官方要求对照、Play Console 字段清单和 Release Ops 行动计划。
