# 今天怎么玩 V0.4 机型适配与首页重构报告

## 本轮测试结论

上一版的问题主要不是功能缺失，而是信息架构和响应式布局：

1. 首页信息太多  
   首页同时承载了品牌、轮播、开局按钮、历史、副本说明、快速开局、会员说明、场景列表，用户第一次打开时会被很多信息打断。

2. 小屏手机首屏压力大  
   固定 440dp 的首页图在小屏上占比过高，入口按钮需要继续下滑才能完整看到。

3. 折叠屏/大屏拉伸感明显  
   卡片、票据和商店内容在大屏上会被横向拉宽，失去参考图里的“精致手机界面/票根”比例。

4. 快速开局不应该挤在首页  
   快速开局本身是一个二级选择流程，放在首页会让主页变成信息列表。

## 已完成修改

1. 首页重构为单独主页
   - 首页只保留品牌标题、电影感轮播和 4 个入口。
   - 入口包括：认真填写开局卡、直接选一个灵感、历史副本、会员商店。
   - 删除首页上的“今日灵感”“会员会得到什么”“可开启副本”等信息栏。

2. 新增独立快速开局页
   - 新页面：`QuickStartScreen`
   - 快速开局卡从首页拆出，进入后再选择具体场景。
   - 增加“周末朋友局，不想只吃饭”预设。

3. 小屏适配
   - 首页根据屏幕高度自动降低轮播图高度。
   - 小屏边距从 22dp 降到 16dp。
   - 首页文字层级降低，避免标题和图片挤压按钮入口。

4. 折叠屏/大屏适配
   - 首页在宽屏下切换为左右双栏：左侧轮播，右侧入口面板。
   - 通用 `SoftCard` 和 `TicketCard` 增加最大宽度限制。
   - 商店页内容在宽屏下居中，不再拉满整屏。
   - 分享卡在宽屏下限制为手机海报比例宽度。
   - 快速开局页在宽屏下改为多卡片横向排列，小屏仍保持单列。

5. 顶栏操作区优化
   - 顶栏右侧操作文字宽度放宽，避免“收藏”等文字被裁切。

6. 版本升级
   - `versionCode`: 4
   - `versionName`: 0.4.0

## 仍待验证

本地没有连接中的真机、模拟器或折叠屏设备，因此本轮完成的是代码层面的响应式适配和构建验证。建议后续重点真机检查：

1. 320dp 宽度小屏：首页入口是否首屏可见。
2. 360dp 常见手机：轮播图和入口面板是否比例舒服。
3. 600dp 小平板/展开折叠屏：首页双栏是否自然。
4. 900dp 以上宽屏：商店页、分享卡是否居中且不拉伸。
5. 横屏：首页双栏是否比单列更好读。
