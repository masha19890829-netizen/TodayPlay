# 今天怎么玩 V0.3 测试与更新总结报告

## 测试视角

本轮继续以“产品测试者 + 热恋中的年轻用户”视角体验：

- 我想马上知道今天怎么玩，不想一直填表。
- 我希望界面更温馨、更像会被收藏的恋爱手帐。
- 我希望知道这个产品为什么值得付费，而不是只看到一个“会员”按钮。

本地仍未检测到连接中的 Android 设备或模拟器，因此验证方式是源码路径审查 + debug APK 构建验证。

## 发现的问题

1. 美术资产太少  
   V0.2 只有一张核心电影感背景图，长时间看会显得重复，首页没有“今天可以有很多种气氛”的感觉。

2. 首屏动效还不够温暖  
   页面有基础浮动装饰，但缺少类似幻灯片、照片切换、慢慢推近的温馨动态。

3. 通关卡仪式感还可以更强  
   V0.2 的通关卡已经可截图，但“完成认证”“盖章感”不足。

4. 商业化入口缺失  
   用户不知道包月能得到什么，也不知道单次付费能买什么，无法理解后续付费价值。

5. 付费表达需要边界清楚  
   MVP 不能让用户误以为已经真实扣费，所以商店必须明确是权益预览。

## 本轮已解决

1. 新增 2 张美术资产
   - `hero_cozy_cafe.png`：温馨咖啡馆约会/开局氛围图。
   - `hero_premium_tickets.png`：会员商店/高级票据氛围图。

2. 首页新增电影感幻灯片
   - 三张图自动轮播。
   - 使用淡入切换和轻微推近效果。
   - 让首页从静态海报变成更温暖的“今日心情相册”。

3. 通关卡新增动态认证章
   - 增加 `GOOD DAY CERTIFIED` 轻微跳动盖章动效。
   - 强化“今天真的通关了”的仪式感。

4. 新增会员商店页
   - 首页增加“会员商店”入口。
   - 新增包月会员权益说明。
   - 新增单次付费包说明。
   - 所有付费按钮当前只展示 Toast，不接真实支付，不会扣费。

5. 付费权益内容
   - 包月会员：更多高级副本、关系模板、纪念日特别局、高级分享卡、历史收藏增强、未来双人共创优先体验。
   - 单次付费：高级今日副本、纪念日特别局、分享卡美化包。

6. 版本升级
   - `versionCode`: 3
   - `versionName`: 0.3.0

## 仍可优化的功能

1. 真正保存分享卡到相册。
2. 历史副本持久化，退出 App 后仍保留。
3. 结果页任务可勾选打卡。
4. 商店接入真实支付前，需要完善退款、自动续费说明、服务条款和隐私合规。
5. 会员权益需要后端或本地权限状态管理。
6. 幻灯片可以继续增加节日、雨天、生日、家庭周末等主题图。
7. 需要真机小屏检查，尤其是商店价格区和通关卡长标题。
